Programación con JavaScript

Practica con +130 ejercicios. Completa 18 PROYECTOS. Aprende HTML, CSS, Node.js, Angular, JSON y más. Todo super claro.

Conoce los conceptos clave del lenguaje de programación que se está comiendo al mundo. Aprende qué es una variable, una función, un objeto y dónde se guardan esos valores. Descubre qué es Scope y cómo se utilizan los loops. Obtén las herramientas para saber cómo tomar decisiones y validar acciones. En este curso, darás el primer paso para empezar tu carrera como desarrollador.

BIENVENIDOS(AS)
  • Este es un programa enfocado en la Programación con JavaScript.En este curso desarrollarás las habilidades técnicas y profesionales necesarias para asegurar tu éxito.
  • Tus clases están diseñadas entre lunes y viernes de 9:30am a 6:00pm. Estas sesiones son una mezcla de sesiones en zoom lideradas por tus instructores, y sesiones de trabajo independiente y en plataformas externas.
  • Mientras trabajas independientemente en estos sitios externos, estarás viendo videos, leyendo artículos, practicando código, respondiendo encuestas, participando en foros de discusión, y/o realizando otras actividades.
LO QUE APRENDERÁS
  • Creación de códigos JavaScript sencillos
  • Escribir pruebas unitarias con Jest
  • Creación y manipulación de objetos y matrices.
COMO APRENDERÁS
  • Sesiones técnicas
  • Sesiones de aplicación de habilidades
  • Sesiones de comunidad
  • Apoyo social y mentoría

HAY 5 MÓDULOS EN ESTE CURSO


JavaScript es el lenguaje de programación que impulsa la web moderna. En este curso, aprenderá los conceptos básicos del desarrollo web con JavaScript. Trabajará con funciones, objetos, matrices, variables, tipos de datos, el DOM (Document Object Model, Modelo de objetos del documento) de HTML y mucho más. Aprenderá a utilizar JavaScript y descubrirá las posibilidades interactivas con las modernas tecnologías de JavaScript. Por último, aprenderá sobre la práctica de probar el código y cómo escribir una prueba unitaria utilizando Jest.

INTRODUCCIÓN A JAVASCRIPT
  1. Aprenderá por qué JavaScript es tan importante para el desarrollo de software.
  2. Obtendrá una visión general de cómo escribir código JavaScript dentro del navegador
  3. Conocerá los operadores más comunes, así como las declaraciones condicionales y los bucles.
COMPONENTES FUNDAMENTALES DE UN PROGRAMA
  1. Aprenderá a utilizar objetos, matrices y funciones.
  2. Aprenderá sobre los métodos incorporados más comunes, y la diferencia entre cadenas indefinidas, nulas y vacías
  3. Explorará la gestión de errores y la programación defensiva.
PARADIGMAS DE PROGRAMACIÓN
  1. Aborda la programación funcional y el paradigma de la programación orientada
  2. Aprenderá qué es el alcance en JavaScript
  3. Explorará las diferencias entre var, let y const.
  4. Aprenderá a utilizar las clases y la herencia en la programación orientada a objetos.
  5. Explorará cómo utilizar la escritura de JavaScript usando características modernas como spread y rest.
  6. Construirá un código que pueda gestionar el DOM y los eventos.
  7. Utilizará JSON en JavaScript.
PRUEBAS
  1. Aprenderá sobre Node.js y Node Package Manager (npm).
  2. Explorará cómo instalar paquetes npm y cómo trabajar con package.json.
  3. Aprenderá sobre las pruebas en JavaScript y codificará una prueba unitaria sencilla en Jest.
EVALUACIÓN CALIFICADA DE FINAL DE CURSO
  1. Sintetizará los conocimientos adquiridos en el curso para crear un código para el “generador de recibos Little lemon”.
  2. Una vez que haya completado las unidades individuales de este módulo, podrá realizar la evaluación calificada. También tendrá la oportunidad de reflexionar sobre el contenido del curso y el camino de aprendizaje que tiene por delante.
